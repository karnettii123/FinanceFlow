# FinanceFlow — Personal Finance Dashboard 💰

A modern, interactive web application for tracking income and expenses with data visualization. This project demonstrates core frontend development skills: DOM manipulation, state management, and working with browser storage.

## 🚀 Key Features
- **Transaction Management:** Add, delete, and view your financial history (CRUD operations).
- **Live Statistics:** Automatic calculation of Total Balance, Total Income, and Total Expenses.
- **Data Visualization:** Dynamic pie charts to visualize spending by category using **Chart.js**.
- **Data Persistence:** All data is saved in **LocalStorage**, so your information stays even after a page refresh.
- **Responsive Design:** Fully functional on desktops, tablets, and smartphones.

## 🛠 Tech Stack
- **HTML5:** Semantic markup for better SEO and accessibility.
- **CSS3:** Custom properties (variables), Flexbox, and Grid for a modern layout.
- **JavaScript (ES6+):** Array methods, Async/Await, and Modular logic.
- **Chart.js:** Integration for professional-grade analytics.
